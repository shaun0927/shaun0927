## 작동 방식

1. Dacon/Kaggle 스크린샷 자동 캡처
2. GPT-4o-mini가 이미지 분석
3. competitions.json 자동 업데이트
4. README.md 생성
5. GitHub에 자동 푸시

## 수동 업데이트 (더 정확함)

```bash
# 1. competitions.json 편집
notepad competitions.json

# 2. README 생성
python update_readme.py

# 3. Git 푸시
git add . && git commit -m "Update" && git push
```

## API 키 설정

## 문제 해결

**API 키 오류**: `.env` 파일 확인
**한글 깨짐**: CMD 인코딩 문제 (실제 파일은 정상)
**Git 푸시 실패**: GitHub 인증 확인
